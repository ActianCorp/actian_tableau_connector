Usage
-----

### Install ODBC driver

Install ODBC Driver from [ESD](https://esd.actian.com/product/Avalanche/Client_Runtime)

NOTE Tableau Desktop is a 64-bit product, it requires a 64-Bit ODBC Driver client installation.
